
 Cinéma du Village - Projet API OMDb

 Description :
Projet étudiant de site web cinéma .

 Fonctionnalités :
- Page d'accueil : Films tendances avec résumés
- Recherche : Barre de recherche en temps réel
- Détails film : Informations complètes d'un film
- Design responsive : Adapté mobile & desktop
  
 Structure du projet :
cinema-omdb-project/
├── index.html      # Page d'accueil
├── search.html     # Page recherche
├── movie.html      # Page détails film
├── style.css       # Styles CSS
├── script.js       # Fonctions principales
├── home.js         # Logique accueil
├── search.js       # Logique recherche
└── detail.js       # Logique détails


   Installation & utilisation :
  Méthode 1 : Avec Live Server 
1. Installer l'extension \`Live Server\` sur VS Code
2. Cliquer droit sur \`index.html\`
3. Sélectionner \`Open with Live Server\`
   
 Méthode 2 : Directement dans le navigateur
1. Ouvrir \`index.html\` dans un navigateur
2. Naviguer entre les pages

 Méthode 3 : Depuis GitHub
\`\`\`bash
 git clone https://github.com/ilyasamaz62-hue/cinema-omdb-project.git
 cd cinema-omdb-project
